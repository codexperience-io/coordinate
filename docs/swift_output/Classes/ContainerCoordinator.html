ContainerCoordinator

open class ContainerCoordinator<T> : Coordinator<T>, HasChildren where T : UIViewController, T : Coordinated

The simplest Coordinator that can have other children Coordinators.

This does everything that Coordinator does plus managing the hierarchy and navigation flow, i.e. how the children are presented and where in the navigation flow. The idea is that you can use it as a switch between any kind of Coordinator.
